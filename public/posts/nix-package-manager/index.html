Troubleshooting:

Error: Programs not showing up in start menu

NIX stores all .desktop files (icons for apps used by the start menu) in /home/$USER/.nix-profile/share/applications, but start menus usually read from /home/$USER/.local/share/applications.

Solution

Make symlinks in /home/$USER/.local/share/applications pointing to /home/$USER/.nix-profile/share/applications

If that doesn’t work, copy the files from /home/$USER/.nix-profile/share/applications to /home/$USER/.local/share/applications
